## Budowanie

Projekt można zbudować używając zainstalowanej już wersji Apache Maven:

```bash
mvn clean install
```

lub wykorzystując Maven Wrapper który nie wymaga wcześniejszej instalacji:

```bash
./mvnw clean install
```
